# 🚀 QuizMaster: The Ultimate Quiz App 🚀

**QuizMaster** is a sleek and engaging quiz application built with modern web technologies. Test your knowledge with a variety of challenging questions in a beautifully designed, responsive interface.

## ✨ Features

-   **🧠 Engaging Questions:** A set of multiple-choice questions to challenge your knowledge.
-   **↔️ Seamless Navigation:** Effortlessly move between questions with "Next" and "Previous" buttons.
-   **📊 Real-time Progress:** Keep track of your progress with a dynamic progress bar.
-   **🏆 Instant Scoring:** See your score update in real-time as you answer questions.
-   **💡 Helpful Hints:** Stuck on a question? Get a hint to help you out.
-   **⏭️ Skip Ahead:** Don't know the answer? Skip the question and come back to it later.
-   **📱 Responsive Design:** Enjoy a flawless experience on any device, from mobile to desktop.

## 🛠️ Technologies Used

-   **HTML5:** The backbone of the application's structure.
-   **CSS3:** For crafting the modern, dark theme and ensuring a responsive layout.
-   **JavaScript (ES6+):** The engine that powers the quiz's logic, interactivity, and dynamic content.

## 🏛️ Architecture

The project is structured to separate concerns, making it easy to understand and maintain.

-   `index.html`: The main HTML file that defines the structure of the quiz application.
-   `style.css`: The stylesheet that provides the visual presentation for the application.
-   `src/`: This directory contains all the JavaScript source code.
    -   `main.js`: The main entry point of the application. It initializes the quiz.
    -   `core/quiz.js`: The core logic of the quiz, including question handling, navigation, and scoring.
    -   `data/questions.js`: Contains the quiz questions and answers.
    -   `utils/dom.js`: A utility file that exports all the DOM elements used in the application.

## 🚀 Getting Started

Ready to take the quiz? Here's how to get started:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/inderbir001/quiz_app.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd quiz_app
    ```
3.  **Open `index.html` in your favorite browser.**

That's it! You're all set to test your knowledge with **QuizMaster**.
